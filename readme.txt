=== imgix plugin for wordpress ===

Requires at least: 3.3

Tested up to: Latest

License: BSD-2

License URI: http://opensource.org/licenses/BSD-2-Clause


This Plugin updates the URLs of all your images to use imgix, for example if the previous URL was http://www.your-site.com/image.jpg .... it should now be converted to http://cdn.your-site.com/image.jpg


== Description ==

Easily use imgix style
